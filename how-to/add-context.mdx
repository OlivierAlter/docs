title: "How-To: Add Context to a Prompt"
description: "Add files, apps, and other context to your prompts"
---

## Steps

1. Drag and drop files onto the notch.
2. Right-click a file and choose Add to Alter.
3. Type `@` in the prompt to add context manually.
