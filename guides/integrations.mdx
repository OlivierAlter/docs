### How to Use Local Tools

<Steps>
  <Step title="Open Tools Manager">
    Click the three dots in Alter → **Tools Manager**
  </Step>
  <Step title="Enable tools">
    In the **Local Tools** section, search and enable tools (Calendar, Maps, etc.)
  </Step>
  <Step title="Use natural language">
    Ask Alter to perform actions:
    - "Send a message to John saying I'm running late"
    - "Add a meeting to my calendar for tomorrow at 2 PM"
    - "Show me my unread emails"
  </Step>
</Steps>

<Note>
  Toggle all tools on/off using the global switch in Tools Manager (Settings > Tools)
</Note>

## x-callback-url Integration

Alter supports the [x-callback-url](https://x-callback-url.com/) protocol for inter-app communication.

### What is x-callback-url?

A standard that lets apps trigger actions in each other using special URLs. Pass data, perform actions, and return seamlessly.

### Triggering Alter from Other Apps

<Steps>
  <Step title="Get the URL">
    In Action Editor, select an action → "URL Callback" section
  </Step>
  <Step title="Copy scheme">
    Copy the `alter://` URL for that action
  </Step>
  <Step title="Use anywhere">
    Paste in Shortcuts, Notes, browser, etc.
  </Step>
</Steps>

**Example:**
```
alter://action/ask-web?input=What+is+Alter+MacOS
```

### Triggering Other Apps from Alter

Create clickable links using target app URL schemes:

| App | Example URL | Action |
|-----|-------------|--------|
| Bear | `bear://x-callback-url/create?title=Demo` | Create note |
| Things | `things://add?title=Task` | Add task |
| Drafts | `drafts://x-callback-url/create?text=Hello` | Create draft |

<Card title="Supported Apps" icon="check-circle">
  Bear, Things, Drafts, OmniFocus, 1Writer, Agenda, and hundreds more
</Card>

  See the [URL Callbacks Guide](/guides/url-callbacks) for complete documentation
