## How do I open Alter?

1. Hover over the notch.
2. Press `Alt+Space`.
3. Hold `Fn` and speak.

![Open Alter](/images/getting-started/open-notch.gif)

See [Getting Started](/guides/getting-started) for more details.

## How do I add my own API key?

1. Open Settings (`Cmd+,`).
2. Go to API Keys.
3. Select provider and enter key.

![Custom API keys](/images/models/custom-api-keys.webp)

See [How-To: Use Your Own API Key](/how-to/use-byok) for detailed instructions.

## How do I add context to a prompt?

- Drag and drop files onto the notch.
- Right-click files and choose Add to Alter.
- Use `@` to add context from the prompt.

![Drag and drop](/images/context/drag-and-drop.gif)

See [The Alter Window](/guides/alter-window#the-context-section) for more details.

## How do I change settings quickly?

- Open Alter and press `Cmd+,`.
- Or open three-dot menu -> Settings.

![Settings menu](/images/settings/settings-menu.png)

See [Alter Settings](/guides/alter-settings) for more details.

## How do I configure dictation?

Settings -> Dictation, then choose processor, language, and replacements.

![Dictation](/images/settings/dictation.webp)

See [Dictation Guide](/guides/dictation) for complete setup instructions.
